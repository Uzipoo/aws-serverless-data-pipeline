# Data Pipeline for Processing CSV Files Using S3 Lambda Glue and QuickSight
Tech Stack: AWS S3, AWS Lambda, AWS Glue, Amazon Athena, Amazon QuickSight, IAM, Python, SQL
Built an end-to-end serverless data pipeline on AWS to automate ingestion, transformation, and visualization of CSV data.
Configured Amazon S3 landing zones for raw and processed datasets with automated event-driven workflows.
Developed AWS Lambda functions triggered by S3 uploads to automate preprocessing and orchestration tasks.
Used AWS Glue Crawlers and ETL jobs to catalog, clean, and transform data for analytics and reporting.
Queried transformed datasets using Amazon Athena and SQL for validation and analysis.
Designed interactive business intelligence dashboards in Amazon QuickSight to visualize KPIs and reporting insights.
Implemented IAM roles and policies to securely manage service permissions and cloud resources.
Troubleshot data pipeline and ETL workflow issues to ensure reliable automation and accurate reporting.
